# Teacher Assistant Pro

Конструктор уроков по ФГОС с использованием ИИ (Cerebras API).

## Структура проекта

```
app/
├── main.py                 # Точка входа приложения Streamlit
├── config.py               # Константы и настройки
├── __init__.py
├── components/             # UI компоненты
│   ├── __init__.py
│   ├── form_components.py  # Компоненты формы создания урока
│   └── result_components.py # Компоненты страницы результатов
├── data/                   # Данные и справочники
│   ├── __init__.py
│   └── subjects.py         # Справочники предметов по уровням школы
└── utils/                  # Утилиты и бизнес-логика
    ├── __init__.py
    ├── prompts.py          # Шаблоны промтов для генерации
    └── generator.py        # Класс генератора материалов
```

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Добавьте API ключ в `.streamlit/secrets.toml`:

```toml
CEREBRAS_API_KEY = "ваш_ключ"
```

## Запуск

```bash
streamlit run app/main.py
```

## Что было оптимизировано

1. **Устранены дубликаты**:
   - Справочники предметов вынесены в отдельный модуль `data/subjects.py`
   - Промты для предметов перенесены в `utils/prompts.py`
   - Константы вынесены в `config.py`

2. **Модульная структура**:
   - UI компоненты разделены на логические блоки
   - Бизнес-логика генерации инкапсулирована в класс `LessonGenerator`
   - Четкое разделение ответственности между модулями

3. **Улучшена читаемость**:
   - Добавлены docstring для всех функций
   - Типизация параметров и возвращаемых значений
   - Логические группы кода объединены в функции

4. **Переиспользование кода**:
   - Helper-функции для получения данных по уровню школы
   - Универсальные компоненты формы
